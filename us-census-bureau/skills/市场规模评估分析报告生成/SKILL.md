# 市场评估报告格式规范

*本 Skill 旨在确保输出的商业战略报告具备咨询级专业水准，逻辑清晰且数据详实。*

## 何时使用

当 Agent 完成了市场规模、人口及饱和度的定量分析，准备生成最终交付报告时，**必须**遵循此规范。

## 核心要求

1. **商业咨询风格**：语言简练、客观，结论先行。
2. **数据可视化描述**：在报告中通过文字描述建议插入的图表类型和内容。
3. **结构化输出**：严格遵守 Markdown 格式。

## 报告生成模板

请忽略中间计算过程，直接输出一份结构化报告。报告必须包含以下章节：

## 美国零售市场规模与饱和度评估报告

### 执行摘要 (Executive Summary)

* **核心现状**：用精炼的语言总结美国零售市场的总体量（规模）与增长速度（增速）。
* **饱和度判断**：给出关于市场饱和度（电商及实体）的关键定性结论（如：高度饱和、快速增长、结构性机会）。
* **战略机会**：列出未来 3-5 年最值得关注的 1-2 个战略切入点。

### 1. 全局市场规模与电商发展态势

* **1.1 总体量与增长性**：
  * 列出最新季度的总销售额、电商销售额。
  * 展示同比（YoY）与环比（QoQ）增长率。
  * *洞察*：分析增长动力主要来自电商拉动还是实体复苏。
* **1.2 线上化进程**：
  * 描述电商渗透率的历史走势。
  * *洞察*：判断线上渠道是否已进入瓶颈期。
  * *(图表建议：在此处描述一张展示“电商渗透率S型曲线”的图表构想)*

### 2. 行业结构与品类格局

* **2.1 供给侧规模**：
  * 报告最新普查年份的零售业总营收规模。
  * 列出市场主体数量（FIRM）及终端网点数量（ESTAB）。
* **2.2 细分赛道扫描**：
  * 如果数据包含细分 NAICS 代码，列出各品类规模。
  * *(数据表格：请插入一个 Markdown 表格，展示关键的行业结构数据)*

### 3. 消费者画像与购买力基础

* **3.1 人口结构特征**：
  * 分析总人口、年龄中位数。
  * 重点关注关键年龄段（如 Z世代、银发族）占比及其对消费偏好的影响。
* **3.2 经济承载力**：
  * 基于贫困率、就业率及教育程度，评估市场的整体消费韧性和支付能力。

### 4. 市场饱和度深度分析

* **4.1 竞争拥挤度**：
  * **每百万人拥有零售企业数**：高密度意味着激烈竞争。
  * **人均零售消费额**：高人均消费意味着成熟市场。
* **4.2 线上/线下饱和度对比**：
  * 对比电商增速与实体零售增速，判断哪一侧更接近天花板。

### 5. 趋势洞察与驱动因素

* **5.1 核心趋势**：识别并阐述当前市场的 3 个主要趋势（例如：全渠道融合、消费两极分化、人口结构带来的品类机会）。
* **5.2 宏观驱动力**：解释技术进步、人口变迁或经济政策如何推动上述趋势。

### 6. 战略建议 (Strategic Recommendations)

* **进入策略**：针对饱和度不同的领域，提出是采取“低价渗透”、“差异化创新”还是“并购进入”的策略。
* **目标客群锁定**：基于画像分析，建议品牌应重点关注的高潜力客群。
* **渠道布局**：基于渗透率分析，提出资源分配建议（如 DTC 优先或全渠道铺设）。

### 附录：数据来源说明

* 简要列出分析所使用的具体 Census 数据表名称及年份。
