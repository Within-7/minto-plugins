# 消费者战略报告格式规范

*本 Skill 旨在确保输出的商业战略报告结构严谨、数据驱动且具有高度的可执行性。*

## 何时使用

当 Agent 已经完成了人口、就业、收入、健康、通勤等多维度数据的交叉分析，并准备向客户（CEO/战略官）交付最终成果时，**必须**参考此规范。

## 核心要求

1. **数据支撑**：报告中的每个关键结论都必须引用具体的统计数据（数值或百分比）作为支撑，严禁模糊描述。
2. **商业洞察**：不要只罗列数字，要解释数字背后的商业含义（"So What?"）。
3. **结构化**：严格遵循以下 Markdown 格式模板。

## 报告生成模板

请输出一份结构清晰、语言专业的报告，包含以下章节：

### 1. 执行摘要 (Executive Summary)

* **针对对象**：CEO/高管级别。
* **内容**：
  * 一句话概述报告的核心发现。
  * 列出 3-5 个最关键的市场机会或风险点。
  * 简述整体战略建议的方向。

### 2. 全景数据洞察 (Comprehensive Data Insights)

*在此部分，请按逻辑顺序展示数据分析的关键发现。*

* **2.1 人口基石**：总人口、年龄结构重心、种族多元化趋势及其对市场多样性的影响。
* **2.2 劳动力与经济底色**：就业稳定性、家庭收入中位数、贫富差距现状分析。
* **2.3 消费约束条件**：贫困率、社会福利依赖度以及无医保人群对非必要消费的挤出效应。
* **2.4 生活方式扫描**：基于职业结构和通勤习惯的消费者日常行为轨迹分析。

### 3. 消费者细分与画像 (Consumer Segmentation & Personas)

*基于数据交叉分析，定义 3-5 个最具代表性的细分市场。每个画像请使用以下格式卡片：*

#### [细分市场名称，例如：富裕的居家办公精英]

* **特征标签**：[例如：高学历 / 科技行业 / 混合办公]
* **数据支撑**：[列出定义该群体的关键数据范围，如收入区间、年龄段]
* **生活方式素描**：[描述其典型的一天及价值观]
* **消费潜力评级**：[高 / 中 / 低] - [简述主要支出方向]

### 4. 战略行动路线图 (Strategic Action Roadmap)

*针对上述画像，提出具体的商业落地建议：*

* **产品策略 (Product)**：应开发或调整什么样的产品以满足特定痛点？
* **定价策略 (Pricing)**：基于购买力评估，建议采用溢价、渗透还是分层定价？
* **渠道策略 (Place)**：基于通勤和生活方式，重点布局线上、CBD 商圈还是社区店？
* **营销策略 (Promotion)**：什么样的价值观主张能打动目标群体？

### 5. 结论 (Conclusion)

* 总结全文，重申数据所指引的最终战略方向。
