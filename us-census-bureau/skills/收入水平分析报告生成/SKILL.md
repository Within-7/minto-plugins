# 收入水平与价格承受力综合报告规范

*本 Skill 旨在确保输出的商业战略报告结构严谨、逻辑闭环，并具有高度的可执行性。*

## 何时使用

当 Agent 完成了收入基准、代际差异、贫困风险、市场分级及弹性预估的所有分析工作后，**必须**加载此 Skill 进行最终文档的生成。

## 核心要求

1. **忽略中间过程**：最终报告中不要包含数据抓取、Python计算代码或中间思考过程。
2. **图表可视化**：使用 Markdown 表格展示数据，使用文本描述（如 `[图表：各年龄段收入分布柱状图]`）来指示关键数据可视化的位置。
3. **商业洞察**：每一部分数据展示后，必须紧跟“战略解读”。

## 报告生成模板

请严格按照以下结构输出报告：

### 1. 执行摘要 (Executive Summary)

* **核心基准**：一句话简报全美家庭收入中位数、平均数及人均收入。
* **关键发现**：概括收入分配形态（偏态性）、最显著的代际差异点以及最脆弱的消费群体。
* **战略核心**：提炼出本报告对企业最重要的3条战略建议（如：定价锚点、核心客群、避险区域）。

### 2. 收入水平分布概览 (Income Distribution Overview)

* **2.1 总体分布图谱**
  * *数据表格*：展示所有家庭在关键收入区间的百分比分布。
  * *形态解读*：详细解释收入中位数与平均数的差距及其揭示的市场结构（例如：是否存在巨大的贫富差距拉动了平均值，这对大众消费品定价意味着什么）。

### 3. 收入差异与市场细分 (Income Disparity & Segmentation)

* **3.1 代际购买力图谱**
  * *数据对比*：展示不同年龄段（25岁以下 vs 45-64岁等）的收入中位数对比。
  * *生命周期洞察*：分析各年龄段的消费潜力（如：年轻人的增长潜力 vs 中年人的现实购买力）。

* **3.2 价格敏感群体警示**
  * *风险数据*：列出贫困率及高风险家庭类型（如单亲母亲、独居老人）的占比。
  * *底线界定*：明确市场的“价格底线”，即该价格以下的商品是满足基本生存需求的。

* **3.3 购买力分级模型 (核心产出)**
  * *分级定义*：
        | 等级名称 | 收入范围 ($) | 家庭占比 (%) | 主要特征 |
        | :--- | :--- | :--- | :--- |
        | 维持型 | < $X | ... | ... |
        | ... | ... | ... | ... |
  * *市场形态*：描述市场构成的形状（如：金字塔型、橄榄型或哑铃型），并分析其对产品定位的影响。

### 4. 收入不平等与弹性分析洞察 (Inequality & Elasticity Insights)

* **4.1 不平等指标集**
  * 列出高低收入比率、家庭结构收入比率。
* **4.2 弹性预判**
  * 基于不平等数据，分析**必需品**与**非必需品/奢侈品**在不同细分市场的潜在需求弹性差异。
  * *例如*：针对高收入组，价格敏感度如何？针对低收入组，基本生活品的价格刚性如何？

### 5. 战略建议与商业影响 (Strategic Recommendations)

*(基于上述所有分析的综合建议)*

* **5.1 精准定价策略**
  * 基于购买力分级，为低、中、高三个市场分别提出定价逻辑（如：撇脂定价 vs 渗透定价的适用性）。
* **5.2 营销与渠道策略**
  * 针对不同代际和收入群体的触达方式建议（例如：高收入老年群体 vs 低收入年轻群体的渠道差异）。
* **5.3 产品创新方向**
  * 基于“未被满足的需求”（如老龄化贫困群体的适老平价产品，或高净值人群的定制服务）提出开发建议。
* **5.4 风险与机遇总结**
  * 总结收入结构变化带来的最大市场风险（如中产阶级萎缩）与潜在的新兴增长点。

### 6. 附录 (Appendix)

* 列出分析所依据的主要数据表名称或编号（如 ACS S1901, B19013 等），以示严谨。
