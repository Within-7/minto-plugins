## 分析执行流程

请按以下步骤执行中间分析（Inner Monologue/Scratchpad）：

### **阶段 1：人口基本结构量化分析（基础层）**

* **目标**：建立人口基线。
* **动作**：
    1. 提取总人口估计值。
    2. 识别主要年龄群体（0-24, 25-64, 65+）占比，判断老龄化趋势。
    3. 计算性别比。
    4. 识别种族/民族的人口规模及占比。

### **阶段 2：社会与家庭结构深度剖析（基础层）**

* **目标**：理解居住单元与社会单元的变化。
* **动作**：
    1. 分析家庭户总数及平均规模变化。
    2. 比较已婚夫妇、单亲、独居家庭的比例。
    3. 识别有未成年子女、老年人及独居长者的比例。
    4. **推论**：预判对住房、日用品及家庭服务的影响。

### **阶段 3：经济能力与教育水平评估（基础层）**

* **目标**：刻画消费潜力和人力资本。
* **动作**：
    1. 分析收入中位数/平均数，识别贫富分化情况。
    2. 分析贫困率。
    3. 统计高学历（本科及以上）比例。
    4. **关联洞察**：总结教育水平如何决定收入潜力。

### **阶段 4：消费市场宏观趋势提炼（基础层）**

* **目标**：扫描市场现状。
* **动作**：
    1. 分析零售市场整体增长态势。
    2. 识别最佳/最差表现行业。
    3. 计算电商销售额占比，评估线上化进程。

### **阶段 5：人口结构与消费趋势关联分析（综合层）**

* **目标**：将“人”的变化与“钱”的流向联系起来。
* **动作**：
    1. **年龄 x 消费**：分析代际消费偏好（如Z世代 vs 银发族）。
    2. **家庭 x 需求**：分析家庭结构变化对房产、家电需求的影响。
    3. **种族 x 多样性**：分析多元化对细分市场的影响。
    4. **收入/教育 x 购买力**：分析消费升级与降级并存现象。
    5. **渠道偏好**：分析不同人群的线上/线下倾向。

### **阶段 6：最终报告生成（交付层）**
