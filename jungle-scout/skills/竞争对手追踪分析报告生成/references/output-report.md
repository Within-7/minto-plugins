# ORF 框架: 竞争对手分析报告标准模板

**最终报告输出要求:** 请将所有分析结果整合为一份结构化的报告，标题为《竞争对手全景追踪与反击策略总案》。报告**必须**包含以下章节和格式：

## 1. 执行摘要 (Executive Summary)

* **核心结论**：用3-5句话高度概括竞品的威胁等级及我方的核心胜算。
* **关键数据速览**：列出竞品预估月收、当前价格、Listing评分、核心流量来源占比。

## 2. 深度诊断：绩效、产品与流量

* **2.1 财务与供应链**：描述Run Rate趋势，量化价格弹性（如“降价10%带来20%排名提升”），评估库存健康度。
* **2.2 产品力透视**：
  * *核心槽点（痛点）*：列出Top 3客诉原因。
  * *真实卖点*：列出用户最认可的Top 3功能。
* **2.3 流量版图**：
  * *核心词占有率*：列出竞品自然排名前3的大词。
  * *广告策略*：评价其广告投放是“激进铺量”还是“保守防御”。

## 3. 战略建模：SWOT与意图

* **竞品生命周期**：明确判定阶段（如：成熟期-利润收割阶段）。
* **SWOT矩阵**：使用Markdown表格展示S、W、O、T四个维度。
* **未来动作预判**：列出竞品最可能采取的3个下一步行动。

## 4. 终局策略：反击行动指南

* **产品侧机会**：基于竞品痛点，明确我方产品的迭代方向（Differentiation）。
* **流量侧打法**：提供一份“进攻关键词表”，列出我方应重点抢占的长尾词或竞品弱势词。
* **分阶段执行计划表**：
  * *Phase 1 (本周)*：[具体战术动作]
  * *Phase 2 (下月)*：[具体战术动作]
  * *Phase 3 (季度)*：[具体战略布局]

## 5. 风险提示与Plan B

* 简述执行策略中可能遇到的风险（如竞品发起价格战）及应对预案。
