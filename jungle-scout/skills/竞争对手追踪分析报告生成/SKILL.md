# 竞争情报与反击行动指南格式规范

*本 Skill 旨在确保输出的竞争分析报告结构严谨、逻辑闭环且具备高度的可执行性。*

## 何时使用

当 Agent 完成了对竞争对手绩效、舆情、流量和战略的分析，准备生成最终交付报告时，**必须**参考此规范。

## 核心要求

1.  **决策导向**：不要罗列数据，要给出“So What”（这意味着什么）和“Now What”（我们要怎么做）。
2.  **结构化**：严格遵循以下 Markdown 模板。
3.  **攻防兼备**：必须同时包含针对竞品弱点的进攻策略和针对竞品优势的防御策略。

## 报告结构概览

1.  **执行摘要 (Executive Summary)**
2.  **竞争对手全景透视 (Deep Dive)**
3.  **战略对决模型 (Strategy SWOT)**
4.  **终局策略：反击行动指南 (Action Plan)**
5.  **风险提示与 B 计划 (Risk & Plan B)**

## 报告生成模板

请直接输出一份结构化、专业、严谨的文本报告，格式如下：

### 1. 执行摘要 (Executive Summary)

*   **核心结论**：用 3-5 句话高度概括竞品的威胁等级及我方的核心胜算。
*   **关键数据速览**：
    *   预估月营收：`$XXX`
    *   市场份额趋势：`上升/持平/下降`
    *   核心流量来源：`自然/广告/站外`

### 2. 竞争对手全景透视 (Deep Dive)

*   **2.1 财务与供应链健康度**
    *   描述销售速度 (Run Rate) 趋势。
    *   量化价格弹性（例如：“降价 10% 通常带来 20% 排名提升”）。
    *   评估库存风险（断货/积压）。

*   **2.2 产品力透视**
    *   *核心槽点（痛点）*：列出 Top 3 客诉原因。
    *   *真实卖点*：列出用户最认可的 Top 3 功能。
    *   *虚实对比*：指出是否存在“功能性欺诈”或“夸大宣传”。

*   **2.3 流量版图**
    *   *核心词占有率*：列出竞品自然排名前 3 的大词。
    *   *广告策略*：评价其广告投放是“激进铺量”还是“保守防御”。

### 3. 战略对决模型 (Strategy SWOT)

*   **竞品生命周期阶段**：`[导入期 / 成长期 / 成熟期 / 衰退期]`（附理由）。
*   **SWOT 矩阵**：（请使用 Markdown 表格展示）

| 维度 | 内容摘要 |
| :--- | :--- |
| **Strengths (优势)** | 竞品难以被攻克的壁垒 |
| **Weaknesses (劣势)** | 竞品的致命缺陷与流量漏洞 |
| **Opportunities (机会)** | 竞品未覆盖的市场/关键词机会 |
| **Threats (威胁)** | 竞品可能发起的反击（如价格战） |

*   **未来动作预判 (Next Moves)**：
    1.  预测动作 A（概率 %）
    2.  预测动作 B（概率 %）

### 4. 终局策略：反击行动指南 (Action Plan)

*(基于洞察提出的可落地建议)*

*   **4.1 差异化进攻 (Attack)**
    *   *产品侧*：针对竞品痛点（Weaknesses），我方产品应如何迭代？
    *   *流量侧*：列出我方应重点抢占的“高潜长尾词”或竞品丢失阵地。

*   **4.2 防御机制建立 (Defense)**
    *   针对竞品优势（Strengths），我方如何建立护城河（如：品牌化、私域沉淀）？

*   **4.3 分阶段执行路线图**
    *   **短期（即刻）**：[具体战术动作，如：调整广告竞价]
    *   **中期（1-3个月）**：[具体战术动作，如：升级产品包装]
    *   **长期（3个月+）**：[具体战略布局]

### 5. 风险提示与 B 计划

*   若竞品发起激进价格战，我方的应对预案。
*   供应链中断风险的应对。
