# 供应链韧性与战略应对报告格式规范

*本 Skill 旨在确保输出的供应链情报简报具备高管级（C-Level）的可读性：逻辑严密、重点突出、图表化思维强。*

## 何时使用

当 Agent 完成了供应链中断的微观提取、宏观分析及风险推演后，准备生成最终交付物时，**必须**严格遵循此规范。

## 核心要求

1.  **高管视角**：避免流水账，直接陈述对业务底线（Bottom Line）的影响。
2.  **图表化思维**：使用 Markdown 表格和列表来展示复杂数据。
3.  **行动导向**：不仅提出问题，必须提供短期和长期的解决方案。

## 报告结构概览

1.  **执行摘要 (Executive Summary)**
2.  **态势感知：微观断点与宏观压力**
3.  **深度影响评估：级联效应推演**
4.  **战略行动框架**
5.  **专家洞察与结语**

## 报告生成模板

请直接输出一份结构化报告，内容必须包含以下章节：

### 1. 执行摘要 (Executive Summary)

*   **核心危机判断**：用3-5句话高度概括，阐述当前供应链面临的最大威胁及其商业影响。
*   **风险雷达 (Risk Radar)**：列出当前紧迫度最高的 Top 3 风险事件。
    *   *格式：[风险等级-高/中/低] 事件名称 - 核心影响简述*

### 2. 态势感知：微观断点与宏观压力 (Situational Awareness)

*   **2.1 关键中断事件全景表**
    请生成 Markdown 表格，包含列：`事件类型`、`发生地点/节点`、`涉及主体/商品`、`预计持续时间`、`状态简述`。

*   **2.2 宏观环境压力仪表盘**
    *   **地缘政治烈度**：(评分1-10) - *简述理由*
    *   **贸易政策紧缩度**：(评分1-10) - *简述理由*
    *   **市场情绪关键词**：(列出3个关键情绪词)

### 3. 深度影响评估：级联效应推演 (Impact Assessment)

*   **3.1 风险传导路径**
    使用文本箭头形式（->）描述冲击如何从上游传递到下游。
    *   *示例：原材料短缺 -> 零部件停产 -> 整车厂交付延迟*

*   **3.2 综合风险量化矩阵**
    请生成 Markdown 表格，包含列：`风险事件`、`宏观背景加成`、`破坏力评分(1-10)`、`影响范围`、`财务/运营冲击预估`。

*   **3.3 受损行业预警**
    明确列出最脆弱的下游行业及其具体痛点。

### 4. 战略行动框架 (Strategic Action Framework)

请构建如下行动建议表，区分战术与战略：

| 职能领域 | 短期战术 (0-3个月) <br> *侧重止损与交付* | 长期战略 (6个月+) <br> *侧重韧性与重构* |
| :--- | :--- | :--- |
| **采购与供应** | (如：激活备选供应商) | (如：实施多元化策略) |
| **库存管理** | (如：建立缓冲库存) | (如：JIT向JIC模式转型) |
| **物流与合规** | (如：改道空运) | (如：数字化供应链投资) |

### 5. 专家洞察与结语 (Expert Insights)

*   **权威声音**：引用文中提及的行业专家或企业领袖的关键观点（Direct Quotes），佐证上述判断。
*   **结语**：一句话总结企业应采取的战略姿态（如“在动荡中寻求敏捷性”）。
