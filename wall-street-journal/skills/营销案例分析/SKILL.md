# 战略级营销洞察报告格式规范

*本 Skill 旨在确保输出的报告具备 WSJ 式的专业、简练、数据驱动且具有前瞻性的风格。*

## 何时使用

当 Agent 完成了营销案例的定性拆解、定量验证和归因分析，准备生成最终交付给 CMO 或投资人的报告时，**必须**参考此规范。

## 核心要求

1.  **决策导向**：结论必须具有行动指导意义，不仅仅是回顾过去。
2.  **去噪音化**：剔除琐碎的执行细节，仅保留战略高度的动作。
3.  **数据支撑**：所有定性观点旁必须附带定量证据（股价、营收、增长率）。

## 报告生成模板

请忽略中间分析过程的思维链展示，直接输出一份结构化、专业的文本报告。报告必须严格包含以下章节：

### 1. 执行摘要 (Executive Summary)

*   **案例定义**：用一句话精准定义该案例的商业价值（例如：“教科书级的品牌年轻化转型”）。
*   **核心数据验证**：列出 3 个最具说服力的关键指标（KPI），直接证明策略有效性。
*   **关键结论**：简要概括 1-3 个核心成功因子（Key Success Factors）。

### 2. 战略全景与市场背景 (Strategic Landscape)

*   **宏观挑战 (The Context)**：简述企业制定策略时面临的 PESTLE 环境压力（如通胀、竞争加剧）。
*   **核心商业叙事 (The Narrative)**：阐述品牌如何重塑公众认知，向资本市场讲述了什么新故事。
*   **策略动作矩阵 (Strategic Moves)**：
    *   *请使用 Markdown 表格展示核心策略：*
    | 维度 (4P) | 关键动作 | 战略意图 |
    | :--- | :--- | :--- |
    | [如：产品/推广] | [关键动作描述] | [旨在解决什么问题] |

### 3. 市场回响与绩效证明 (Market Validation)

*   **资本市场反应**：描述股价对营销动作的反应（如：公告后股价涨幅、市盈率重估），并结合分析师的情感倾向（引用关键 Quote）。
*   **财务与份额表现**：通过营收增长、同店销售或市场份额数据证明业务基本面的改善。
*   **数据透视表**：
    *   *请提供核心数据的对比：*
    | 关键指标 | 策略实施前 | 策略实施后/当前 | 变化幅度 |
    | :--- | :--- | :--- | :--- |
    | [指标名] | [数值] | [数值] | [百分比] |

### 4. 深度归因与启示 (Deep Dive & Takeaways)

*   **成功归因分析**：
    *   **内因 vs 外因**：明确区分策略带来的增长与环境红利。
    *   **核心驱动逻辑**：用简练的语言描述“A 动作如何导致了 B 结果”。
*   **首席顾问建议 (Advisory Note)**：
    *   **可借鉴点**：具体的、可操作的策略建议，适用于同类企业。
    *   **可复制性评级**：[高/中/低] —— 并说明理由（是否依赖特殊资源）。
    *   **避坑指南**：实施类似策略时需警惕的风险点。

### 5. 结论 (Conclusion)

*   基于当前策略，预测品牌未来的挑战或机会。
