# 商业战略报告格式规范：定价趋势洞察

*本 Skill 旨在确保输出的定价策略报告具有高度的专业性、逻辑性和可操作性，适合直接呈报给企业高管。*

## 何时使用

当 Agent 完成了对 WSJ 数据的全景分析（阶段 1-4），准备生成最终交付文档时，**必须**遵循此规范。

## 核心要求

1.  **洞察驱动**：不要简单罗列新闻，要提供“So What”（这对业务意味着什么）。
2.  **结构清晰**：严格遵循以下 Markdown 结构。
3.  **语言风格**：使用战略咨询顾问风格（客观、精炼、使用专业术语如“结构性通胀”、“定价权”、“尾部风险”）。

## 报告生成模板

请忽略中间的思考过程，直接输出一份结构化的文本报告。报告必须包含以下章节：

### 1. 执行摘要 (Executive Summary)

*   **核心结论**：用 3-5 行文字，高度概括当前定价环境的本质（例如：“由能源成本驱动的结构性滞胀风险”）。
*   **关键信号**：列出 3 个最重要的市场信号，必须结合“数据事实”与“背后原因”（格式：信号 - 数据 - 归因）。

### 2. 市场全景诊断 (360° Diagnosis)

*   **2.1 量化热力图 (Quantitative Heatmap)**
    *   以 Markdown 表格形式，列出关键资产（宏观指数、核心大宗商品、关键汇率）的最新价格、变动方向及异常状态评级。
*   **2.2 深度归因分析 (Root Cause Analysis)**
    *   详细解释导致当前价格局面的核心逻辑。明确指出是供应端问题（如物流、产能）还是需求端问题。
*   **2.3 市场情绪温度 (Market Sentiment)**
    *   描述市场参与者的心理状态（如：企业正在积极去库存、市场对通胀持观望态度）。引用权威专栏或央行观点作为支撑。

### 3. 趋势研判与情景预测 (Outlook & Scenarios)

*   **3.1 基准情景 (6-12个月)**
    *   基于驱动力和情绪分析，预测最可能的价格走势（如：通胀延续、触顶回落或剧烈波动）。
*   **3.2 风险情景 (Black Swans)**
    *   指出可能破坏基准预测的尾部风险（如地缘政治升级、政策急转弯）。
*   **3.3 权威预期汇总**
    *   引用华尔街主流机构或分析师的具体预测共识。

### 4. 商业战略建议 (Strategic Recommendations)

*针对企业客户（如制造业、进出口商），提供具体的行动指南：*

*   **4.1 供应链与采购策略**
    *   建议动作（例如：锁定长期合同以防涨价 / 按需采购以待跌价）。
*   **4.2 定价与销售策略**
    *   企业应如何调整终端价格？（例如：利用通胀预期提价 / 通过促销清库存 / 剥离低毛利产品）。
*   **4.3 风险与金融对冲**
    *   针对汇率或原材料波动的具体对冲思路。

### 5. 附录：关键数据支撑

*   简要列出支撑上述结论的 3-5 个最关键的数据点来源或 WSJ 文章标题。
