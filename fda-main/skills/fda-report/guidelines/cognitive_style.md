# 认知风格与格式标准 (Cognitive Style & Formatting Standards)

## 1. “顾问之声” (The Consultant's Voice)
*   **目标:** 保持权威性，避免被动语态。
*   **错误示范:** “看起来可能存在风险。”
*   **正确示范:** “数据显示系统性失效的可能性很高 (置信度: 高)。”

## 2. 视觉人体工程学 (Visual Ergonomics)
*   **表格:** 任何超过 2 项的对比都必须使用表格。
*   **粗体:** 加粗 *洞察*，而不是标签。
    *   *错误:* **风险:** 高
    *   *正确:* 风险: **高 (严重影响)**
*   **引用块:** 使用 `>` 来展示高层级摘要或关键警告。

## 3. 逻辑结构 (Logical Structure)
*   **金字塔原理:**
    1.  **情境 (Situation):** 事实是什么？
    2.  **冲突 (Complication):** 问题在哪里？
    3.  **问题 (Question):** 我们该怎么做？
    4.  **答案 (Answer):** 核心建议 (放在最前面)。

## 4. 数据引用规范 (Data Citation)
*   严禁模糊引用 (如 "据报道...")。
*   必须使用具体 ID (如 "Ref: Recall #Z-1234-2023")。
