# 安全成本与定价策略报告格式规范

*本 Skill 旨在确保输出的战略咨询报告结构严谨、数据详实且具有高管决策视角。*

## 何时使用

当 Agent 完成了安全成本拆解、TCO 建模和价格敏感度分析后，准备生成最终交付报告时，**必须**参考此规范。

## 核心要求

1.  **高管视角**：结论先行，重点关注财务影响（TCO, ROI, 利润率）。
2.  **数据可视化**：在文本中使用 Markdown 表格或图表描述来展示关键数据对比。
3.  **逻辑连贯**：遵循“成本 —> 效益 —> 价格 —> 战略”的逻辑链条。

## 报告结构概览

1.  **执行摘要 (Executive Summary)**
2.  **模块一：安全投入的总体拥有成本 (TCO) 与效益评估**
3.  **模块二：安全成本对价格策略的量化影响**
4.  **战略结论与建议 (Strategic Conclusions & Recommendations)**

## 报告生成模板

请忽略中间计算过程的冗长展示，直接输出一份结构化、专业、严谨的文本报告。报告必须包含以下章节：

### 1. 执行摘要 (Executive Summary)

*   **核心发现 (Top 3 Insights)**：提炼关于合规成本结构、非合规风险敞口（具体金额）以及市场价格敏感度的三个最关键结论。
*   **关键建议 (Actionable Recommendations)**：简述企业应采取的成本优化方向和定价策略（如“溢价策略”或“成本吸收策略”）。

### 2. 模块一：安全投入的总体拥有成本 (TCO) 与效益评估

*   **2.1 TCO 财务结构分解**：
    *   使用表格展示预防性、检测性、纠正/风险三类成本的分布与占比。
    *   *重点分析*：指出哪一类成本占比最高，以及 CPSC 合规要求对该成本的驱动作用。
*   **2.2 安全投入的价值量化 (ROI)**：
    *   报告基于历史罚款和召回数据估算的“潜在风险敞口”金额。
    *   展示安全投入带来的风险降低百分比及显性 ROI 指标。
    *   *图表描述*：描述“安全投入成本 vs. 潜在风险成本”的平衡曲线及最佳投入点。

### 3. 模块二：安全成本对价格策略的量化影响

*   **3.1 价格调整的必要性测算**：
    *   明确列出：**单位产品新增安全成本**、**维持利润率所需的最小价格涨幅（%）**。
*   **3.2 价格调整的情景影响预测**：
    *   对比分析“完全转嫁成本”与“竞争对齐定价”两种情景下，企业**净利润**与**市场份额**的预测变化。
    *   *表格要求*：使用 Markdown 表格对比两种情景的关键财务指标（销量、收入、利润）。
*   **3.3 市场竞争反应**：
    *   结合竞争对手历史数据，分析市场对价格调整的可能反应（如价格战风险或跟进涨价）。

### 4. 战略结论与建议 (Strategic Conclusions & Recommendations)

*(基于 TCO 和价格模型提出的可执行建议)*

*   **4.1 成本优化策略**：针对 TCO 模型中的高成本项（如强制性测试或潜在罚款风险），提出具体的降低成本或转移风险的建议。
*   **4.2 定价行动路线图**：
    *   **建议转嫁率**：明确建议企业将多少比例的安全成本转嫁给消费者（0% - 100%）。
    *   **目标价格设定**：给出具体的建议市场进入价格区间。
*   **4.3 竞争优势构建**：建议如何利用“高合规标准”作为营销卖点，以支撑定价策略（将安全成本转化为品牌溢价）。
