## Design Principles

### Visual Hierarchy
1. **Primary Elements**: Titles and key metrics (32-40px)
2. **Secondary Elements**: Subtitles and section headers (24-28px)
3. **Tertiary Elements**: Body text and details (16-20px)
4. **Emphasis Elements**: Highlighted data and calls-to-action (orange #F85d42)

### Color Psychology
- **Orange (#F85d42)**: Action, energy, emphasis
- **Deep Blue (#556EE6)**: Trust, professionalism, stability
- **Green (#34c38f)**: Growth, success, positive trends
- **Black (#000000)**: Authority, elegance, headers

### Spatial Relationships
- Consistent 40px padding on all content slides
- 24px grid gaps for multi-column layouts
- 16px vertical spacing between elements
- Balanced white space for readability

## Common Issues and Solutions

### 1. Content Overflow
**Problem**: Too much content for single slide
**Solution**: 
- Split into multiple slides
- Use progressive disclosure
- Create collapsible sections
- Optimize text density

### 2. Color Inconsistency
**Problem**: Colors not displaying correctly
**Solution**:
- Verify CSS variable definitions
- Check for conflicting inline styles
- Ensure proper color format (hex codes)
- Test across different browsers

### 3. Download Failures
**Problem**: html2canvas not working
**Solution**:
- Verify external script loading
- Check for CORS issues with images
- Ensure proper canvas dimensions
- Provide fallback options

### 4. Navigation Issues
**Problem**: Slides not switching properly
**Solution**:
- Verify slide indexing logic
- Check event listener attachments
- Ensure proper CSS class management
- Test keyboard event handling

## Best Practices for Complex Presentations

### Multi-section Presentations
1. **Consistent Section Headers**: Use same styling across all sections
2. **Clear Transitions**: Visual cues for section changes
3. **Progressive Disclosure**: Reveal information strategically
4. **Visual Anchors**: Consistent elements for orientation

### Data-Heavy Presentations
1. **Prioritize Key Metrics**: Highlight most important data points
2. **Use Progressive Disclosure**: Break complex data into digestible chunks
3. **Apply Consistent Scales**: Maintain proportional relationships
4. **Provide Context**: Include reference points and benchmarks

### Image-Rich Presentations
1. **Optimize Image Quality**: Balance quality with file size
2. **Consistent Image Treatment**: Apply same borders, shadows, effects
3. **Strategic Placement**: Position images for maximum impact
4. **Alt Text Inclusion**: Ensure accessibility compliance
